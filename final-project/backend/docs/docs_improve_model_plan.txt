plan for improvements in the model:

Phase 1: The Preprocessing & Normalization Layer
Goal: Solve obfuscation (Problem 3) and misspellings (Problem 5).

Action: Before the text ever reaches the BERT model, it passes through a cleaner. This layer will:

Remove artificial separators (dashes, periods between letters).

Collapse repeating characters (e.g., "שששששלווווום" -> "שלום").

Replace common obfuscation symbols (asterisks, @ symbols) with generic characters or remove them to stitch words back together.

Phase 2: The Chunking Engine (Sliding Window) - done, does not really work well yet - storing this version here locally
Goal: Solve the long text and contrast/mixed sentiment issues (Problems 1 & 4).

Action: Instead of feeding a massive paragraph to the model at once, we split the text into sentences or overlapping chunks. We evaluate each chunk individually. If any single chunk crosses the toxicity threshold, the entire text is flagged.

Phase 3: The "Zero-Tolerance" Heuristics Cache - done, need to add many many more words here
Goal: Catch severe cultural context and extreme curses instantly without relying on the AI's probability scores (Problems 2 & 3).

Action: Implement a fast, regex-based blocklist for severe slurs, known modern slang phrases (e.g., references to Oct 7th), and exact-match misspellings. If a text hits this layer, it is instantly blocked, saving the processing time of running the BERT model.

Phase 4: Dynamic Thresholding & Model Tuning - done
Goal: Maximize accuracy without over-blocking.

Action: As we noticed, 0.7 is slightly too high for obfuscated Hebrew. We will implement a dynamic threshold (e.g., 0.65 for Hebrew, 0.7 for English) or evaluate if we need to swap the underlying model to a dedicated Hebrew NLP model (like DictaBERT) if the multilingual one still falls short after the first three phases.
